DESIGN STANDARDS SUMMARY
================================================================================

COLORS
- Dark baseline: #0a0a0f, #12121a, #1a1a24
- Text: #e4e4e7 (primary), #71717a (muted), #3f3f46 (dimmed)
- Accent: Per-project (replaces cyan #22d3ee)
- Semantic: Green (#4ade80) success, Amber (#fbbf24) warning, Red (#ef4444) error

TYPOGRAPHY
- Display/Headings: Space Grotesk
- Body/Content: Space Grotesk
- UI/Labels/Code: IBM Plex Mono or JetBrains Mono
- No pure white (use #e4e4e7), no pure black (use #0a0a0f)

SPACING
- Scale: xs(4px), sm(8px), md(16px), lg(24px), xl(32px), 2xl(40px), etc.
- Padding: Consistent use of spacing scale
- Gaps: 4px, 8px, 16px, 24px minimum
- Margins: Use spacing scale, not magic numbers

ROUNDED CORNERS
- Maximum: 8px (lg)
- Typical: 2px (xs) or 4px (sm)
- Avoid: Over-rounded elements

BORDERS
- Width: 1px only
- Color: #1f1f2e (subtle) or #27272a (hover) or accent color (active)
- No thick borders, keep subtle

ANIMATIONS
- Entrance: 800-1200ms with easeOutQuint
- Hover: 200-300ms with easeOut
- Transitions: 300ms with easeOut
- Stagger: 100-150ms between elements
- GPU-accelerate: Use transform and opacity only

ACCESSIBILITY
- Contrast: 4.5:1 for text (WCAG AA)
- Focus: Always visible
- Keyboard: Tab, Enter, Escape navigation
- Screen Reader: Semantic HTML, ARIA labels

RESPONSIVE
- Mobile-first approach
- Breakpoints: 768px (md), 1024px (lg)
- Touch targets: 44x44px minimum
- No horizontal scroll

USAGE EXAMPLES
================================================================================

APPLY PROJECT THEME:
```tsx
import { applyProjectTheme } from '@/design/themes';

useEffect(() => {
  applyProjectTheme('truthbounty');  // Purple theme
}, []);
```

USE DESIGN TOKENS:
```tsx
import { semanticTokens, baseTokens } from '@/design/tokens';

<div style={{
  color: semanticTokens.colors.text.primary,
  padding: baseTokens.spacing.lg,
  gap: baseTokens.spacing.md,
}}>
```

BUILD COMPONENT:
```tsx
import { buttonPattern } from '@/design/components';

export function Button({ variant = 'primary', size = 'md', children }) {
  return (
    <button className={`
      ${buttonPattern.baseClasses}
      ${buttonPattern.sizes[size]}
      ${buttonPattern.variants[variant].base}
    `}>
      {children}
    </button>
  );
}
```

ANIMATE ELEMENTS:
```tsx
import { animationTiming, easing, getStaggerDelay } from '@/design/animations';

{items.map((item, i) => (
  <div
    key={i}
    style={{
      animation: `slideInUp ${animationTiming.slowTrans} ${easing.easeOutQuint}`,
      animationDelay: getStaggerDelay(i, 100),
    }}
  >
    {item}
  </div>
))}
```
